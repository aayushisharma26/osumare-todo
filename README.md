1. **Clone the repository:**

   ```bash
   git clone https://github.com/aayushisharma26/osumare-todo
   cd osumare-todo
   ```

2. **Install dependencies:**

   Using npm:

   ```bash
   npm install
   ```

   Or using yarn:

   ```bash
   yarn install
   ```

3. **Start the development server:**

   Using npm:

   ```bash
   npm start
   ```

   Or using yarn:

   ```bash
   yarn start
   ```

   The application will run at `http://localhost:3000`.
